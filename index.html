目前所有的Buff分成三種，先稱之為1類、2類、以及3類。

1類的Buff僅有覺醒的KillUp和WaveUp兩個類別。共用100 % 傷害 、100 % 跑速、50 % 防禦、50 % 技能傷害、50 % 技能跑速、100 % 技能防禦的上限

2類的Buff有覺醒的DeadPlusCID PowerDrain SuperSkillFull2 ManaSpend ExtraResourse WaveSupport WaveSupportAll ManaPower AutoMotion FlagAdvance FlagAdvancePlus KillUpPlus，以及技能的DAMAGED_ATTACK DAMAGED_SHOOT FLAG_ATTACK HEAL_UNION SUPPORT_AREA。共用-50%~100%傷害、-50%~100%技能傷害、-50%~50%跑速、-50%~100%防禦、0%~50%爆擊、-150%~100%爆擊傷害、-50%~100%攻速的界限限制。

3類則是所有其他的，無上限。

技能的MANA_CYCLE，覺醒的StopUp、Position、SuperSkillFull、ChangeFormUp的總攻速倍率範圍為0.5~2倍。

技能的MANA_CYCLE，覺醒的MoveUp、StopUp、Position、LowHP、HighHP、FullHP、FlagUp、SuperSkillFull、ChangeFormUp、HeroOne、HoldMana、ManaCountUp、BossBattle的總暴擊率範圍為0~0.5。

覺醒的StrengthWeapon、StrengthHome、TargetStageAll、ManaSlotUp、StrengthJob2、Backup的總暴擊率範圍為0~0.5。

技能的MANA_CYCLE，覺醒的Position、StopUp、LowHP、HighHP、SuperSkillFull、ChangeFormUp、HeroOne、HoldMana、ManaCountUp的總暴擊傷害加成範圍為-1~2。

覺醒的TargetStageAll、ManaSlotUp、StrengthJob2的總暴擊傷害範圍為-1~2。

覺醒的ManaHealer的總治療範圍倍率範圍為0.5~2。

覺醒的ManaHealer、SuperSkillFull的總治療量倍率範圍為0~2

普攻、治療、殺敵的超必殺增加範圍為-100~100單位

擊退力代表一秒會被擊退幾單位，被擊中後會在一秒內遞減至0，因此總移動距離是擊退力/2

一格代表1.5單位

白骨化: 30秒
黑暗: 60秒
毒: 15次, 4.2秒一次, 每次5%
衰弱: 15次, 4.2秒一次, 每次5%
封技: 30秒
