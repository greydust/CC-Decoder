1. 下載Nox App Player
2. 將該模擬器開啟Root, 把lucky-patcher-6-2-2.apk裝進去
3. 打開它, 選Toolbox -> Patch to Android -> Disable .apk Signature Verification, 再按Apply
4. 弄完以後就可以把修改過的兩個CC apk和NoxDataProcessor.apk裝進去
5. 分別打開兩個CC, 跑完登入以後切掉, 再打開NoxDataProcessor, 等到它變成Complete, 就更新完成了
6. 轉出來的檔案在C:\Users\使用者名稱\Nox_share\Other
